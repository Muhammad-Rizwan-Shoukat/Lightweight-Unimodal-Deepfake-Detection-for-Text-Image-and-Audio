# 🚀 Deepfake Detection System

### Lightweight Unimodal Deepfake Detection for Text, Image, and Audio

---

## 📖 Overview

This project presents a research-grade unimodal deepfake detection framework capable of detecting AI-generated content in text, image, and audio domains.

The system combines:

* ⚡ Lightweight scratch-built architectures
* 🧠 Pretrained deep learning models
* 🔄 Optimized preprocessing pipelines

Designed for high accuracy, low computational cost, and real-time applicability.

---

## ⭐ Key Contributions

* 🎯 Custom lightweight models:

  * VerbaFake (Text)
  * EchoFake (Audio)
  * PixelFake (Image)

* ⚡ Efficient architectures suitable for resource-constrained environments

* 📈 High-performance detection across all modalities

* 🛡️ Robust preprocessing and augmentation strategies

## 🤖 Models

The framework includes both pretrained and custom-built lightweight models:

* Attention-basedBiLSTM
* Efficient-CapsNet
* VerbaFake (Scratch_Built)

### 🖼️ Image

* EfficientNetV2-S
* MobileNetV2
* PixelFake (Scratch_Built)

### 🎙️ Audio

* Xception
* DenseNet201
* EchoFake (Scratch_Built)
